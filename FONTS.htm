Fonts

Here is the deal: Fonts go a long way! I am aiming to have very few images, so to add some "pop," I have made use different fonts and different styles of those fonts. That said I also want to make using the fonts as esay as possible!.

3 Main Typefaces

I use Verdana as the main font, plus I added two Google Fonts: Oswald and Roboto. As such, the class names: "verdana", "oswald", and "roboto" can be used to style fonts.

Styling fonts that way got tiresome, so I made shortcuts using bold tags, like this:

  • <b class="o"> = Oswald
  • <b class="r"> = Roboto
  • <b class="m"> = Monospaced
If you don't enter anything the default font is Verdana.

Other Selectors

Italics, Bold, and Underline are done the normal way. (Using: <i>, <b>, and <u>.)

Modifiers

Font sizing can be changed up with:

  • class="tiny"
  • class="small"
  • class="big"
  • class="huge"
The default font size is 16px.

Special Cases

NOTES and TIPS:
<div class="note">This will be smaller!</div> results in:

This will be smaller!

However, the "note" class also decreases line-spacing (in case it wraps), so it is not merely a smaller font.
